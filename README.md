# backend-app

## Requirements
- Python 3.7+
- Flask
- Flask-RESTful
- Flask-SQLAlchemy
- Flask-Marshmallow
- Marshmallow-SQLAlchemy

## Setup

1. Clone the repository
2. Create a virtual environment and activate it
3. Install the dependencies
4. Run the application
5. Run the tests
